# 2095. Delete The Middle Node Of A Linked List

### O(n) - TS with two passes

## Complexity

- Time: $$O(n)$$
- Space: $$O(1)$$

## Submission

Runtime: 5 ms · Beats 40.81%
Memory: 95.58 MB · Beats 99.10%

## Code

```typescript
function deleteMiddle(head: ListNode | null): ListNode | null {
  if (!head?.next) return null;

  let dummy: ListNode | null = head;
  let length = 0;

  while (dummy) {
    dummy = dummy.next;
    length++;
  }

  dummy = head;
  const stop = Math.ceil(length / 2) - 1;

  for (let i = 0; i < stop; i++) {
    dummy = dummy!.next;
  }

  dummy!.next = dummy!.next?.next ?? null;
  return head;
}
```
