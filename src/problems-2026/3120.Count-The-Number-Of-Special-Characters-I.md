# 3120. Count The Number Of Special Characters I

### O(n) - TS with Set()

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

Runtime: 1 ms · Beats 93.75%

## Code

```typescript
function numberOfSpecialChars(word: string): number {
  const codes = word.split("").map((char) => char.charCodeAt(0));
  const set = new Set(codes);
  let res = 0;

  for (let i = 0; i < 32; i++) {
    if (set.has(i + 65) && set.has(i + 97)) {
      res++;
    }
  }

  return res;
}
```
