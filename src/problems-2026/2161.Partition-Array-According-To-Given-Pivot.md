# 2161. Partition Array According To Given Pivot

### O(n) - TS with partitioning

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

Runtime: 21 ms · Beats 86.67%
Memory: 85.62 MB · Beats 93.33%

## Code

```typescript
function pivotArray(nums: number[], pivot: number): number[] {
  const less: number[] = [];
  const more: number[] = [];
  const equal: number[] = [];

  nums.forEach((num) => {
    if (num < pivot) {
      less.push(num);
    } else if (num > pivot) {
      more.push(num);
    } else {
      equal.push(num);
    }
  });

  return [...less, ...equal, ...more];
}
```
