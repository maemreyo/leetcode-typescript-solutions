# 3612. Process String With Special Operations I

### O(n^2) - TS with simulation

## Complexity

- Time: $$O(n^2)$$
- Space: $$O(n)$$

## Submission

## Code

```typescript
function processStr(s: string): string {
  let res = "";
  const array = s.split("");

  array.forEach((c) => {
    switch (c) {
      case "*":
        res = res.slice(0, s.length - 1);
        break;
      case "#":
        res += res;
        break;
      case "%":
        res = res.split("").reverse().join("");
        break;
      default:
        res += c;
        break;
    }
  });

  return res;
}
```
