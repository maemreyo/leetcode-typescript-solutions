# 2558. Take Gifts From The Richest Pile

### O((n + k) log n) - TS with MaxHeap

## Complexity

- Time: $$O((n + k) \log n)$$
- Space: $$O(n)$$

## Submission

Runtime: 16 ms · Beats 77.78%
Memory: 58.13 MB · Beats 44.44%

## Code

```typescript
import { MaxHeap } from "@datastructures-js/heap";

const getFloorSqrt = (num: number) => Math.floor(Math.sqrt(num));

function pickGifts(gifts: number[], k: number): number {
  const heap = new MaxHeap<number>();
  gifts.forEach((gift) => heap.insert(gift));

  for (let i = 0; i < k; i++) {
    heap.insert(getFloorSqrt(heap.pop()!));
  }

  return heap.toArray().reduce((sum, val) => sum + val, 0);
}
```
