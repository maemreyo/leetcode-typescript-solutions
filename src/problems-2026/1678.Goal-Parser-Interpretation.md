# 1678. Goal Parser Interpretation

### O(n) - TS with single-pass pointer skipping

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

- Runtime: 40 ms · Beats 92.31%
- Memory: 55.58 MB · Beats 38.46%

## Code

```typescript
function interpret(command: string): string {
  let res = "";
  let i = 0;

  while (i < command.length) {
    if (command[i] === "G") {
      res += "G";
      i += 1;
    } else if (command[i + 1] === ")") {
      res += "o";
      i += 2;
    } else {
      res += "al";
      i += 4;
    }
  }

  return res;
}
```
