# 1967. Number Of Strings That Appear As Substrings In Word

### O(n * m) - TS with reduce and includes

## Complexity

- Time: $$O(n \cdot m)$$
- Space: $$O(1)$$

## Submission

- Runtime: 0 ms · Beats 100.00%
- Memory: 55.98 MB · Beats 32.14%

## Code

```typescript
function numOfStrings(patterns: string[], word: string): number {
  return patterns.reduce(
    (acc, patern) => acc + Number(word.includes(patern)),
    0
  );
}
```
