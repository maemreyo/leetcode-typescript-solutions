# 3925. Concatenate Array With Reverse

### O(n) - TS with spread and array reverse

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

- Runtime: 1 ms · Beats 94.12%
- Memory: 61.07 MB · Beats 5.88%

## Code

```typescript
function concatWithReverse(nums: number[]): number[] {
  return [...nums, ...[...nums].reverse()];
}
```
