# 2942. Find Words Containing Character

### O(n * m) - TS with reduce and includes

## Complexity

- Time: $$O(n \cdot m)$$
- Space: $$O(n)$$

## Submission

- Runtime: 1 ms · Beats 82.69%
- Memory: 59.03 MB · Beats 67.31%

## Code

```typescript
function findWordsContaining(words: string[], x: string): number[] {
  return words.reduce<number[]>((res, word, i) => {
    if (word.includes(x)) res.push(i);
    return res;
  }, []);
}
```
