# 2553. Separate The Digits In An Array

### TS Simple One-Liner

## Complexity

- Time: $$O(n \cdot d)$$
- Space: $$O(n \cdot d)$$

## Submission

Runtime: 2 ms · Beats 100.00%

Memory: 58.91 MB · Beats 82.14%

## Code

```typescript
function separateDigits(nums: number[]): number[] {
  return nums.join("").split("").map(Number);
}
```
