# 3498. Reverse Degree Of A String

### O(n) - TS with single pass and char code math

## Complexity

- Time: $$O(n)$$
- Space: $$O(1)$$

## Submission

- Runtime: 2 ms · Beats 36.36%
- Memory: 57.30 MB · Beats 100.00%

## Code

```typescript
function reverseDegree(s: string): number {
  let res = 0;
  for (let i = 0; i < s.length; i++) {
    const reversedPos = 123 - s.charCodeAt(i);
    res += reversedPos * (i + 1);
  }
  return res;
}
```
