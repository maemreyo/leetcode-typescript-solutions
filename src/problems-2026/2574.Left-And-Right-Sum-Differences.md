# 2574. Left And Right Sum Differences

### O(n) - TS with prefix and suffix sum arrays

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

- Runtime: 3 ms · Beats 51.08%
- Memory: 59.59 MB · Beats 15.29%

## Code

```typescript
function leftRightDifference(nums: number[]): number[] {
  const left = new Array<number>(nums.length).fill(0);
  const right = new Array<number>(nums.length).fill(0);
  const res = new Array<number>(nums.length).fill(0);

  for (let i = 1; i < nums.length; i++) {
    left[i] = left[i - 1] + nums[i - 1];
  }

  for (let i = nums.length - 2; i >= 0; i--) {
    right[i] = right[i + 1] + nums[i + 1];
  }

  for (let i = 0; i < nums.length; i++) {
    res[i] = Math.abs(left[i] - right[i]);
  }

  return res;
}
```
