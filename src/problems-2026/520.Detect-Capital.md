# 520. Detect Capital

### O(n) - TS with counting

## Complexity

- Time: $$O(n)$$
- Space: $$O(1)$$

## Submission

Memory:  · Beats

## Code

```typescript
// A-Z = ASCII 65-90
function isCapitalAscii(char: string): boolean {
  const code = char.charCodeAt(0);
  return code >= 65 && code <= 90;
}

function detectCapitalUse(word: string): boolean {
  let capitalized = 0;
  for (let i = 0; i < word.length; i++) {
    if (isCapitalAscii(word[i])) capitalized++;
  }

  // Case 1: all caps → "USA"
  // Case 2: no caps → "leetcode"
  // Case 3: first only → "Google"
  return (
    capitalized === word.length ||
    capitalized === 0 ||
    (capitalized === 1 && isCapitalAscii(word[0]))
  );
}
```
