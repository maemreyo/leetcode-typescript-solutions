# 2965. Find Missing And Repeated Values

### O(n^2) - TS with counting array

## Complexity

- Time: $$O(n^2)$$
- Space: $$O(n^2)$$

## Submission

- Runtime: 1 ms · Beats 100.00%
- Memory: 58.90 MB · Beats 82.22%

## Code

```typescript
function findMissingAndRepeatedValues(grid: number[][]): number[] {
  const n = grid.length;
  const total = n ** 2;
  const counts = new Array(total + 1).fill(0);

  for (const row of grid) {
    for (const val of row) counts[val]++;
  }

  let repeated = 0;
  let missing = 0;

  for (let i = 1; i <= total; i++) {
    if (counts[i] === 2) repeated = i;
    else if (counts[i] === 0) missing = i;
  }

  return [repeated, missing];
}
```
