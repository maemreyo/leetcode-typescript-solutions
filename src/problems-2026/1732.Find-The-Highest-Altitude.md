# 1732. Find The Highest Altitude

### O(n) - TS with prefix sum

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Code

```typescript
function largestAltitude(gain: number[]): number {
  const dp: number[] = [...gain];

  for (let i = 1; i < dp.length; i++) {
    dp[i] += dp[i - 1];
  }

  const max = Math.max(...dp);

  return max > 0 ? max : 0;
}
```
