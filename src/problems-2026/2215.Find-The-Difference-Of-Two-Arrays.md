# 2215. Find The Difference Of Two Arrays

### O(n + m) - TS with Set() lookups

## Complexity

- Time: $$O(n + m)$$
- Space: $$O(n + m)$$

## Submission

- Runtime: 10 ms · Beats 56.32%
- Memory: 63.07 MB · Beats 48.85%

## Code

```typescript
function findDifference(nums1: number[], nums2: number[]): number[][] {
  const set1 = new Set(nums1);
  const set2 = new Set(nums2);
  const missing1: number[] = [];
  const missing2: number[] = [];

  for (const n of set1) if (!set2.has(n)) missing1.push(n);
  for (const n of set2) if (!set1.has(n)) missing2.push(n);

  return [missing1, missing2];
}
```
