# 1833. Maximum Ice Cream Bars

### O(n log n) - TS with sorting

## Complexity

- Time: $$O(n \log n)$$
- Space: $$O(n)$$

## Submission

Runtime: 128 ms · Beats 14.29%

## Code

```typescript
function maxIceCream(costs: number[], coins: number): number {
  const sorted = [...costs].sort((a, b) => a - b);
  let res = 0;

  for (const cost of sorted) {
    if (cost > coins) break;
    coins -= cost;
    res++;
  }

  return res;
}
```
