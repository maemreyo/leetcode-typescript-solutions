# 1108. Defanging An Ip Address

### O(n) - TS with replaceAll

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

- Runtime: 45 ms · Beats 40.63%
- Memory: 55.63 MB · Beats 10.94%

## Code

```typescript
function defangIPaddr(address: string): string {
  return address.replaceAll(".", "[.]");
}
```
