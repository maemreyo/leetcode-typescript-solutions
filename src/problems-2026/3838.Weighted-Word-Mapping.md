# 3838. Weighted Word Mapping

### O(n * m) - TS with reduce

## Complexity

- Time: $$O(n \cdot m)$$
- Space: $$O(n)$$

## Submission

Runtime: 2 ms · Beats 69.66%
Memory: 59.04 MB · Beats 25.84%

## Code

```typescript
function mapWordWeights(words: string[], weights: number[]): string {
  return words
    .map((word) =>
      String.fromCharCode(
        122 -
          (word
            .split("")
            .reduce((acc, c) => acc + weights[c.charCodeAt(0) - 97], 0) %
            26)
      )
    )
    .join("");
}
```
