# 482. License Key Formatting

### O(n) - TS with string reversal

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

Runtime: 10 ms · Beats 52.27%
Memory: 63.91 MB · Beats 21.59%

## Code

```typescript
function licenseKeyFormatting(s: string, k: number): string {
  const reversed = s
    .replaceAll("-", "")
    .toUpperCase()
    .split("")
    .reverse()
    .join("");

  const groups: string[] = [];

  for (let i = 0; i < reversed.length; i += k) {
    groups.push(reversed.slice(i, i + k));
  }

  return groups.join("-").split("").reverse().join("");
}
```
