# 1979. Find Greatest Common Divisor Of Array

### O(n + m) - TS with brute-force GCD loop

## Complexity

- Time: $$O(n + m)$$
- Space: $$O(1)$$

## Submission

- Runtime: 0 ms · Beats 100.00%
- Memory: 55.93 MB · Beats 69.81%

## Code

```typescript
function findGCD(nums: number[]): number {
  const min = Math.min(...nums);
  const max = Math.min(...nums);

  for (let i = min; i > 1; i--) {
    if (min % i === 0 && max % i === 0) {
      return i;
    }
  }

  return 1;
}
```
