# 3289. The Two Sneaky Numbers Of Digitville

### O(n) - TS with Set duplicate tracking

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

- Runtime: 1 ms · Beats 90.91%
- Memory: 58.24 MB · Beats 30.91%

## Code

```typescript
function getSneakyNumbers(nums: number[]): number[] {
  const seen = new Set<number>();
  const sneaky: number[] = [];

  nums.forEach((num) => {
    if (seen.has(num)) sneaky.push(num);
    seen.add(num);
  });

  return sneaky;
}
```
