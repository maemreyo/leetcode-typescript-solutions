# 3668. Restore Finishing Order

### O(n + m) - TS with Set filter

## Complexity

- Time: $$O(n + m)$$
- Space: $$O(m)$$

## Submission

- Runtime: 0 ms · Beats 100.00%
- Memory: 58.60 MB · Beats 68.67%

## Code

```typescript
function recoverOrder(order: number[], friends: number[]): number[] {
  const set = new Set([...friends]);
  return [...order].filter((friend) => set.has(friend));
}
```
