# 3794. Reverse String Prefix

### O(n) - TS with string reversal

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

- Runtime: 2 ms · Beats 50.00%
- Memory: 56.13 MB · Beats 96.67%

## Code

```typescript
function reversePrefix(s: string, k: number): string {
  return s.slice(0, k).split("").reverse().join("") + s.slice(k);
}
```
