# 3754. Concatenate Non-Zero Digits And Multiply By Sum I

### O(log n) - TS with replaceAll and reduce

## Complexity

- Time: $$O(\log n)$$
- Space: $$O(\log n)$$

## Submission

- Runtime: 1 ms · Beats 60.00%
- Memory: 56.49 MB · Beats 93.33%

## Code

```typescript
function sumAndMultiply(n: number): number {
  const s = n.toString().replaceAll("0", "");
  const sum = s.split("").reduce((acc, d) => acc + Number(d), 0);
  const x = Number(s);
  return x * sum;
}
```
