# 3701. Compute Alternating Sum

### O(n) - TS with reduce

## Complexity

- Time: $$O(n)$$
- Space: $$O(1)$$

## Submission

- Runtime: 0 ms · Beats 100.00%
- Memory: 57.44 MB · Beats 89.58%

## Code

```typescript
function alternatingSum(nums: number[]): number {
  return nums.reduce((res, num, i) => res + (i % 2 === 0 ? num : -num), 0);
}
```
