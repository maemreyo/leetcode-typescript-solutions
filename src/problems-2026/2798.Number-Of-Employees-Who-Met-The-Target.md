# 2798. Number Of Employees Who Met The Target

### O(n) - TS with filter

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

- Runtime: 0 ms · Beats 100.00%
- Memory: 57.99 MB · Beats 26.32%

## Code

```typescript
function numberOfEmployeesWhoMetTarget(
  hours: number[],
  target: number
): number {
  return hours.filter((h) => h >= target).length;
}
```
