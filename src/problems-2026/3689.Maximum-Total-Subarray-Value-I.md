# 3689. Maximum Total Subarray Value I

### O(n) - TS with math

## Complexity

- Time: $$O(n)$$
- Space: $$O(1)$$

## Submission

Runtime: 3 ms · Beats 100.00%
Memory: 67.76 MB · Beats 66.67%

## Code

```typescript
function maxTotalValue(nums: number[], k: number): number {
  const max = Math.max(...nums);
  const min = Math.min(...nums);
  return (max - min) * k;
}
```
