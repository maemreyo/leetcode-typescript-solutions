# 2418. Sort The People

### O(n log n) - TS with index sort by height

## Complexity

- Time: $$O(n \log n)$$
- Space: $$O(n)$$

## Submission

- Runtime: 6 ms · Beats 80.00%
- Memory: 61.26 MB · Beats 36.00%

## Code

```typescript
function sortPeople(names: string[], heights: number[]): string[] {
  return heights
    .map((height, i) => [height, i])
    .sort((a, b) => b[0] - a[0])
    .map(([_, i]) => names[i]);
}
```
