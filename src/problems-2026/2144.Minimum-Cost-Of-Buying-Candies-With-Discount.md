# 2144. Minimum Cost Of Buying Candies With Discount

### O(n log n) - TS with sorting

## Complexity

- Time: $$O(n \log n)$$
- Space: $$O(n)$$

## Submission

Runtime: 1 ms · Beats 94.12%
Memory: 57.50 MB · Beats 23.53%

## Code

```typescript
function minimumCost(cost: number[]): number {
  cost.sort((a, b) => b - a);

  return cost
    .filter((_, i) => i % 3 !== 2)
    .reduce((acc, curr) => acc + curr, 0);
}
```
