# 3945. Digit Frequency Score

### O(log n) - TS with string spread and reduce

## Complexity

- Time: $$O(\log n)$$
- Space: $$O(\log n)$$

## Submission

- Runtime: 0 ms · Beats 100.00%
- Memory: 56.94 MB · Beats 90.08%

## Code

```typescript
function digitFrequencyScore(n: number): number {
  return [...String(n)].map(Number).reduce((acc, curr) => acc + curr, 0);
}
```
