# 2974. Minimum Number Game

### O(n \log n) - TS with Heap

## Complexity

- Time: $$O(n \log n)$$
- Space: $$O(n)$$

## Submission

Runtime: 23 ms · Beats 5.26%
Memory: 59.79 MB · Beats 10.53%

## Code

```typescript
import { MinHeap } from "@datastructures-js/heap";

function numberGame(nums: number[]): number[] {
  const heap = new MinHeap<number>();
  const res: number[] = [];

  nums.forEach((num) => {
    heap.insert(num);
  });

  while (!heap.isEmpty()) {
    const a = heap.pop()!;
    const b = heap.pop()!;

    res.push(b);
    res.push(a);
  }

  return res;
}
```
