# 3467. Transform Array By Parity

### O(n log n) - TS with map and toSorted

## Complexity

- Time: $$O(n \log n)$$
- Space: $$O(n)$$

## Submission

- Runtime: 1 ms · Beats 94.34%
- Memory: 58.42 MB · Beats 43.40%

## Code

```typescript
function transformArray(nums: number[]): number[] {
  return nums.map((num) => (num % 2 === 0 ? 0 : 1)).toSorted((a, b) => a - b);
}
```
