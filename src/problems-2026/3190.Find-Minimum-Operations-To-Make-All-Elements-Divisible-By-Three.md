# 3190. Find Minimum Operations To Make All Elements Divisible By Three

### O(n) - TS with filter

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

- Runtime: 0 ms · Beats 100.00%
- Memory: 58.32 MB · Beats 12.20%

## Code

```typescript
function minimumOperations(nums: number[]): number {
  const min = nums.filter((num) => (num %= 3));
  return min.length;
}
```
