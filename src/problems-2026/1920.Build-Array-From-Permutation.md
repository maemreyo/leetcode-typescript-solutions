# 1920. Build Array From Permutation

### O(n) - TS with single-pass index mapping

## Complexity

- Time: $$O(n)$$
- Space: $$O(n)$$

## Submission

- Runtime: 1 ms · Beats 62.20%
- Memory: 57.78 MB · Beats 98.78%

## Code

```typescript
function buildArray(nums: number[]): number[] {
  const ans = new Array<number>(nums.length);

  for (let i = 0; i < nums.length; i++) {
    ans[i] = nums[nums[i]];
  }

  return ans;
}
```
