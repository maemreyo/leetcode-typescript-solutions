# 2500. Delete Greatest Value In Each Row

### O(n log n) - TS with sorting

## Complexity

- Time: $$O(m \cdot n \log n)$$
- Space: $$O(m \cdot n)$$

## Submission

Runtime: 7 ms · Beats 84.21%
Memory: 56.00 MB · Beats 100.00%

## Code

```typescript
function deleteGreatestValue(grid: number[][]): number {
  const sortedGrid = grid.map((row) => [...row].sort((a, b) => a - b));
  let res = 0;

  for (let i = 0; i < grid[0].length; i++) {
    let max = 0;
    for (let j = 0; j < grid.length; j++) {
      max = Math.max(max, sortedGrid[j].pop()!);
    }
    res += max;
  }

  return res;
}
```
